一、热力学基础：判断反应能否自发
1. 基本概念
体系与环境：敞开体系（物质 + 能量交换）、封闭体系（仅能量交换）、孤立体系（无交换）。
状态函数：只与始末态有关，与路径无关（如内能 U、焓 H、熵 S、吉布斯自由能 G）。
热力学第一定律：ΔU = Q + W（能量守恒，Q 吸热为正，W 环境对体系做功为正）。
2. 焓与反应热
焓变 ΔH：恒压下反应的热效应，ΔH < 0放热，ΔH > 0吸热。
盖斯定律：总反应焓变 = 各分步反应焓变之和（与路径无关）。
标准生成焓：由最稳定单质生成 1mol 化合物的焓变，用于计算反应焓。
3. 熵与混乱度
熵 S：描述体系混乱度，混乱度越大，S 越大（气体 > 液体 > 固体；温度升高，S 增大）。
熵增原理：孤立体系中，自发反应总朝熵增（ΔS 总 > 0） 方向进行。
4. 吉布斯自由能：反应自发判据
核心公式：ΔG = ΔH - TΔS
ΔG < 0：反应自发正向进行；
ΔG = 0：反应达到平衡状态；
ΔG > 0：反应非自发（逆向自发）。
自发规律：
放热（ΔH<0）+ 熵增（ΔS>0）：任何温度都自发；
吸热（ΔH>0）+ 熵减（ΔS<0）：任何温度都不自发；
放热 + 熵减：低温自发；吸热 + 熵增：高温自发。
二、化学平衡：反应的限度
1. 平衡的本质
正反应速率 = 逆反应速率，体系各物质浓度不再随时间变化（动态平衡，非反应停止）。
2. 平衡常数 K
表达式：对反应 aA + bB ⇌ cC + dD，
K = [C]^c[D]^d / ([A]^a[B]^b)（固体、纯液体不写入）。
K 意义：K 越大，正向反应越完全；K 只与温度有关，与浓度、压强无关。
3. 化学反应等温式
ΔG = ΔG° + RTlnQ
Q：反应商（浓度 / 分压实时比值）；
Q <K：正向自发；Q = K：平衡；Q> K：逆向自发。
4. 勒夏特列原理（平衡移动原理）
改变条件（浓度、压强、温度），平衡向减弱该改变的方向移动：
增大反应物浓度：平衡正向移动；
升温：平衡向吸热方向移动；
加压：平衡向气体分子数减少的方向移动。
三、化学动力学：反应的快慢
1. 反应速率
定义：单位时间内反应物浓度减少或生成物浓度增加，v = Δc/Δt。
速率方程：对基元反应 aA + bB → 产物，v = k[A]^a[B]^b（k 为速率常数，与温度、催化剂有关）。
2. 反应级数
速率方程中浓度幂之和，决定浓度对速率的影响：
零级反应：v 与浓度无关，半衰期恒定；
一级反应：v 与浓度一次方成正比，半衰期 t₁/₂ = ln2/k（与浓度无关）。
3. 活化能与反应速率
活化能 Eₐ：反应物到达活化状态所需的最小能量，Eₐ越小，反应越快。
阿伦尼乌斯公式：k = A·e^(-Eₐ/RT)，温度升高、催化剂降低 Eₐ，均使 k 增大，速率加快。
4. 影响反应速率的因素
浓度：浓度增大，单位体积活化分子数增多，速率加快；
温度：温度升高，活化分子百分数大幅增加，速率显著加快；
催化剂：降低活化能，同等条件下加快正、逆反应速率，不改变平衡常数。
四、核心逻辑总结
热力学：解决 “反应能不能发生、方向与限度”，不涉及时间；
动力学：解决 “反应多快、如何加速”，关注反应路径；
二者结合：自发反应（ΔG<0）未必快，非自发反应可通过外界做功（如电解）实现。
